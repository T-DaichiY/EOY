Graphics
Bitmap · Vector · Colour depth · Resolution · File formats · Visual communication
Bitmap images
A bitmap stores an image as a fixed grid of coloured squares called pixels. When enlarged, pixels stretch and can become blurry or blocky.
  • Best for photographs, screenshots, scanned documents, and pixel art.
  • Formats include JPG, PNG, BMP, and GIF.
  • File size depends on width, height, and colour depth.
File size (bits) = Width(px) × Height(px) × Colour depth(bpp)
Bitmap vs vector
Vector images
A vector stores shapes as mathematical coordinates, curves, lines, and fills. It can be resized without losing quality.
  • Best for logos, icons, diagrams, maps, and illustrations.
  • Common formats include SVG, AI, and EPS.
  • Not suitable for real photographs.
File formats

.jpg

  • Lossy
  • Photos
  • No transparency

.png

  • Lossless
  • Transparency
  • Logos/icons

.bmp

  • Uncompressed
  • Very large

.gif

  • Animation
  • 256 colours

Networks
Devices · Bandwidth · Topologies · Cloud · URLs · DNS
Network devices
  • Router: connects different networks, such as home network to internet.
  • Switch: sends data only to the intended device.
  • Hub: broadcasts data to every device.
  • Modem: converts ISP signals into usable network data.

DNS and URLs
DNS translates domain names into IP addresses. A URL contains parts such as subdomain, domain, top-level domain, path, and file name.

AI & Machine Learning
Traditional AI · Generative AI · ML · Neural Networks · LLMs · Ethics
Traditional AI vs Generative AI

Traditional AI

  • Specific tasks
  • Classifies, filters, predicts
  • Spam filter, chess engine

Generative AI

  • Creates new content
  • Text, image, music, code
  • ChatGPT, DALL·E
Machine learning
  • Supervised learning uses labelled data.
  • Unsupervised learning finds patterns in unlabelled data.
  • Reinforcement learning learns through rewards and penalties.
